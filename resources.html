Архитектура проекта EasyAccess

EasyAccess работает как браузерное расширение, которое динамически модифицирует веб-страницы для повышения их доступности:

Основные компоненты системы

  • База данных: Хранилище пакетов модификаций и пользовательских настроек
  • Клиентская часть: Интерфейс для выбора и установки модификаций
  • Серверная часть: Управление пакетами модификаций и аутентификация пользователей
  • Вспомогательные сервисы: Анализ документов и обработка запросов

Как работает EasyAccess

Расширение анализирует исходный код веб-страницы и применяет к нему CSS и JavaScript инъекции из выбранных пользователем пакетов модификаций, преобразуя неадаптированный сайт в доступную версию.
